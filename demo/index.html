About this explorer

Qwanqwa (phonetic spelling of ቋንቋ, QQ for short) is a language metadata toolkit for NLP: identifiers, scripts, speakers, geographic data, and traversable relationships across languoids, scripts, and regions.

In QQ, language-like entities are called languoids ↗: this includes dialects, macro-languages, and language families, not just individual languages. Coverage differs across entities.

You can search for languoids, scripts, and regions, inspect their properties, copy values, and explore related entities in the graph.

We combine and normalize data from the sources listed below. Properties include:

  • Identifiers such as BCP-47, ISO 639, Glottocode, Wikidata, and related code systems
  • Multilingual names, speaker counts, endangerment data, scripts, and geographic coverage
  • Traversable relationships across language families, scripts, and regions
  • And more! See the GitHub repo ↗ and the Python package ↗ for more complete programmatic access.
